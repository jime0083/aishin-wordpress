- [ ] `https://` で全ページが開き、ブラウザに鍵マークが表示される

- [ ] www あり/なしが片方に統一されてリダイレクトされる
- [ ] `/wp-admin/` に強固なパスワードでログインできる（初期パスワードは変更済み）
- [ ] Limit Login Attempts Reloaded が有効
- [ ] WordPress本体・プラグインの自動更新が有効
- [ ] Lightsail自動スナップショットが有効
- [ ] AWS Budgets の請求アラートが設定済み
